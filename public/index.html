It looks like you are in the wrong place!
This server was designed to be used with POST requests!

There are currently 3 endpoints to use:

  • /on

    Turns the lights on.

  • /off

    Turns the lights off.

  • /

    Accepts JSON data using this format:
    { hex: "01ABEF" }
    This will set the color of the lights.
